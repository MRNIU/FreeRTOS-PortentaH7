# FreeRTOS-PortentaH7

GSoC 2020 Arduino project.

[Portenta H7](https://www.arduino.cc/pro/hardware/product/portenta-h7) is a high performance board released by [Arduino](https://www.arduino.cc).

## Usage

Compile:
`make MCU=M7`

Upload:
`make MCU=M7 upload`

Clean build file:
`make MCU=M7 clean`

`MCU=M7` or `MCU=M4`
